# Computer Vision Projects with OpenCV.js

This repository contains a collection of interactive **computer vision projects** built with **OpenCV.js**, JavaScript, HTML, and webcam/image processing techniques.

The projects demonstrate real-time image filtering, feature extraction, template matching, thresholding, edge detection, and webcam-based processing directly in the browser.

## Projects Included

- **Canny Edge Detection**
- **Gaussian Blur**
- **Image Convolution**
- **Template Matching**
- **Thresholding**
- **Webcam Template Matching with OpenCV.js**

## Tech Stack

- **HTML**
- **JavaScript**
- **OpenCV.js**
- **OpenCV**
- **Python** (for local server support)

## Features

- Real-time browser-based computer vision demos
- Image filtering and enhancement
- Edge detection and thresholding
- Template matching workflows
- Webcam-based OpenCV.js processing
- Hands-on experiments with core computer vision concepts

## Project Structure

- `cannyEdge.html` / `cannyEdge.js`
- `gaussianBlur.html` / `gaussianBlur.js`
- `imgConvolution.html` / `imgConvolution.js`
- `templateMatching.html` / `templateMatching.js`
